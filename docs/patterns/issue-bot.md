> Issue 机器人是一种自动化工具，通常集成在软件开发和项目管理平台中，用于处理和跟踪项目中的问题报告（Issues）。这些问题报告可以包括软件缺陷、功能请求、任务、建议等。

## 示例：Intellij IDEA

Intellij 相关的 IDEA 会在异常时：

1. 提供创建 Issue 的选项
2. 如果 issue 已经存在，会通过 comments 的方式创建。
